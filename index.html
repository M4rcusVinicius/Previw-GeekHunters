Este artigo tem como objetivo te ensinar a desenvolver uma barra de navegação responsiva com animação utilizando apenas um CSS simples. Melhorando a navegação e o aproveitamento dos clientes do seu site sem muita dificuldade.

Neste artigo você vamos utilizar conceitos fundamentais de FlexBox, que vai nos ajudar a organizar espacialmente os itens da barra de navegação. As animações serão feitas através da função :hover do CSS, mudando a disposição dos elementos da NavBar quando o usuário passar o mouse por cima.

Implementando ao HTML

Primeiramente precisamos criar uma classe contendo toda a nossa barra de navegação, o HTML possui uma tag específica chamada <nav>, em seguida vamos criar uma lista <ul>, contendo cada item <li> com uma um link <a> contendo o texto e o ícone.

<nav class="navbar"> <ul class="navbar-nav"> <li class="nav-item"> <a href="#" class="nav-link"> <span class="nav-text">Config.</span> <i class="fa fa-cog"></i> </a> </li> </ul> </nav>

Essa é uma estrutura básica de um item da NavBar, você pode adicionar quantos itens quiser, apenas acrescentando mais itens na lista "li".

Adicionando ícones

Os ícones utilizados são importados de Font Awesome, com mais de 5000 ícones personalizáveis para que você possa escolher o melhor para o seu projeto.

Atenção: Não se esqueça de colocar o script do Fonts Awesome no seu <head>

<script src="https://kit.fontawesome.com/7128c13a32.js" crossorigin="anonymous"></script>

Formatando o CSS

Agora precisamos utilizar o CSS para remover as margens padrão no navegador.

body { font-family: 'Times New Roman', Times, serif; margin: 0; padding: 0; }

Para posicionar a barra de navegação, utilize o position: fixed; para manter a barra fixa na tela, mesmo que naveguemos na página para cima e para baixo. A altura possui 100vh, ou seja, 100% da viewport (tela).

.navbar { top: 0; width: 70px; height: 100vh; position: fixed; background-color: #212325; transition: width 600ms ease; }

Customizando os itens

Delimite o espaço ocupado pela lista <ul class="navbar-nav"> e utilize o flexbox para alinhar verticalmente os itens da lista. Utilizamos margin-top: auto; para posicionar o último item da lista no fundo.

.navbar-nav { list-style: none; padding: 0; margin: 0; display: flex; flex-direction: column; align-items: center; height: 100%; } .nav-item { width: 100%; text-align: center; font-size: 20px; letter-spacing: 0.3ch; width: 100%; } .nav-item:last-child { margin-top: auto; } .nav-logo { background-color: #0e1114; font-weight: bold; }

Utilize novamente o Flexbox para alinhar o texto e o ícone horizontalmente, para isso digite display: flex; na classe pai (nav-link). Tire a formatação padrão da lista e dos links com o text-decoration: none;.

.nav-link { display: flex; align-items: center; height: 50px; text-decoration: none; color: #cad9e7; transition: 200ms; } .nav-link i { width: 2rem; min-width: 2rem; margin-left: 16px; transition: 600ms; }

Posicione o texto para fora da página do lado esquerdo, para que ele não esteja visível.

.nav-text { display: inline; position: absolute; left: -999px; }

Animando a NavBar

Aplicamos o transition anteriormente para que os movimentos sejam mais lentos e possamos ver a animação. Para que a animação aconteça quando o mouse passar por cima da NavBar, vamos utilizar o .navbar:hover.

/* Aumenta a NavBar para 200px */ .navbar:hover { width: 200px; } /* Move o texto de -999px para 15px a partir do caanto esquerdo */ .navbar:hover .nav-text { left: 15px; } /* Move os itens para a direita */ .navbar:hover .nav-item i{ margin-left: 150px; }

Coloque um hover para mudar a cor dos itens quando o mouse passar por cima.

.nav-link:hover { background: #292d31; color: rgb(142, 181, 255); }

Adicionando animações nos itens

Como o posicionamento dos itens são dinâmicos, os itens se posicionam automaticamente ao alterar o tamanho de um dos ítens. Pensando nisso podemos criar uma foto de perfil que com um layout um pouco diferente dos outros itens.

Adicione um item da lista do HTMl com classes específicas para o logo, não se esqueça de colocar uma imagem chamada logo.png na mesma pasta do HTML, ou mude o atributo src.

<li class="nav-logo"> <a href="#" class="nav-logo-link"> <img class="nav-logo-png" src="logo.png"> <h1 class="nav-logo-text">Nome do usuário</h1> </a> </li>

Formate a cor, o tamanho e o posicionamento dos itens no CSS. O texto surge como se estivesse crescendo pois colocamos o tamanho da fonte igual a 0px e vamos aplicar um hover com tamanho igual a 20px.

.nav-logo { background-color: rgb(20, 20, 22); padding: 10px 0 10px 0; width: 100%; } .nav-logo-png { width: 38px; height: 38px; transition: 600ms; } .nav-logo-text { color: #cad9e7; transition: 600ms; font-size: 0px; } .nav-logo-link { text-decoration: none; display: flex; flex-direction: column; align-items: center; }

Por fim vamos adicionar um hover simples, para que os elementos aumentar o tamanho dos elementos

.navbar:hover .nav-logo-text { font-size: 20px; } .navbar:hover .nav-logo-png { width: 100px; height: 100px; }
